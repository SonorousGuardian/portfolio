### The Overview
Block-Buddy is a user-friendly web application designed to streamline residential property management. It simplifies the interaction between landlords and tenants through a centralized platform.

### Key Features
* **Tenant Management:** Enabled secure tenant registration and automated maintenance requests.
* **Rent Tracking:** Implemented transparent rent tracking systems to monitor payments effectively.
* **Admin Dashboard:** Built an analytics dashboard for property owners to view occupancy rates and financial statistics.

### Tech Stack

* <i class="fa-solid fa-flask" style="color: #fff;"></i> **Backend:** Flask (Python)
* <i class="fa-solid fa-database" style="color: #39ff14;"></i> **Database:** SQLite
* <i class="fa-brands fa-html5" style="color: #e34f26;"></i> **Frontend:** HTML5, CSS3, JavaScript
